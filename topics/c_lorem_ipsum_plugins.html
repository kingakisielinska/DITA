Lorem Ipsum Plugins

You can install various plugins to create greeking text.

Table 1. Plugins:
Program Description
MS Word From 2007 onwards typing =lorem(i) or =lorem(i, j) in MS Word creates greeking text--don't forget to hit the enter key; i and j need to be integers, where i stands for the amount of desired paragraphs and j numbers the sentences per paragraph. j has a default value of 3 and a maximum set to 6665.
Open Office An extension named Magenta Lorem ipsum generator is available at services.openoffice.org. Once installed, an icon will be shown in a toolbar below the menu bar, with the option to insert a number of paragraphs, the maximum is set to 25. >> screenshots
Google Docs Lorem Generator: A plugin for Chrome browsers that allows designers and webmaster to generate Lorem Ipsum text from the sidebar without leaving Google Docs.
